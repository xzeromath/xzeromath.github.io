## git 사용법
- git init
  - 폴더에 .git을 만들고 이 폴더가 git을 사용한다는 것을 알려줌
- git clone
  - github과 같은 장소에서 내용을 복사해옴. 이때 저장소와 연결까지 되는 것 같음
- git add .
- git commit -m 'text'
- git push
