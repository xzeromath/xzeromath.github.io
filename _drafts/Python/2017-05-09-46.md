## vscode 사용
.vscode/settings.json 에 다음과 같이 파이썬 파일의 위치를 지정하면 된다.
``` python
"python.linting.pylintEnabled": false,
"python.pythonPath":"/Users/ckn/craw/bin/python3"
```
